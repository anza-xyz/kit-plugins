Add `@solana/kit-plugin-signer` package with plugins for setting up `payer` and `identity` signers on Kit clients. Each plugin comes in three variants (`signer*`, `payer*`, `identity*`) to set both roles, only the payer, or only the identity. Includes `signer`, `generatedSigner`, `generatedSignerWithSol`, `signerFromFile`, and `airdropSigner` plugin families.
